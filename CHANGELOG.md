# Changelog

所有重要的项目变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，

## [0.1.0] - 2026-04-15

### 新增

- 无操作自动息屏（默认 10 分钟）
- 息屏前友好提醒（右下角 Toast 通知 + 30 秒倒计时）
- 智能时段控制（仅下班后启用，工作时间自动禁用）
- 一键息屏（托盘右键菜单）
- 开机自启动支持
- 精致设置界面（现代化卡片 UI）
- 暂停/恢复检测功能
- 多显示器支持
- GitHub Actions 自动构建发布流程
